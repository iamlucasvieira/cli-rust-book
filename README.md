- **echor**: Rust version of echo


## Running the CLI's
First you need to have Rust installed. 

Then you can run
```bash
cargo run -p [CLI NAME] -- [COMMANDS/ARGS]
```
Example: `cargo run -p echor -- "Hello, World!"`
